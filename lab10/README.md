# Лабораторная работа 10: Адаптивный лендинг с Bootstrap 5

## Описание

Адаптивный лендинг для компании "БизнесЛанч", доставляющей бизнес-ланчи в офис. Разработан с использованием фреймворка Bootstrap 5.

## Структура проекта

```
lab10/
├── landing.html       # Главная страница лендинга
├── style.css          # Кастомные стили
├── images/            # Папка с изображениями
│   └── hero-lunch.svg # SVG иллюстрация для hero секции
└── README.md          # Этот файл
```

### 4. Contact Section (Обратная связь)

  - Адрес

## Адаптивность

Страница адаптирована для следующих breakpoints:

- **Desktop** (≥992px): Полная версия с многоколоночной сеткой
- **Tablet** (768px-991px): Двухколоночная сетка для преимуществ
- **Mobile** (≤767px): Одноколоночная сетка, уменьшенные заголовки
- **Small Mobile** (≤575px): Оптимизированные кнопки и шрифты

### Responsive Grid Examples:
```html
<!-- Hero Section -->
<div class="row align-items-center">
  <div class="col-lg-6">...</div>  <!-- 50% на десктопе, 100% на мобильных -->
  <div class="col-lg-6">...</div>
</div>

<!-- Features -->
<div class="row g-4">
  <div class="col-md-6 col-lg-4">...</div>  <!-- 33% на десктопе, 50% на планшете, 100% на мобильных -->
</div>

<!-- Contact -->
<div class="row g-5">
  <div class="col-lg-7">...</div>  <!-- 58% на десктопе, 100% на мобильных -->
  <div class="col-lg-5">...</div>  <!-- 42% на десктопе, 100% на мобильных -->
</div>
```

## Особенности реализации

1. **Самостоятельная верстка**: Все блоки созданы с нуля без использования готовых шаблонов
2. **Bootstrap Grid System**: Активно используется во всех секциях
3. **Semantic HTML**: Использованы семантические теги (nav, section, footer)
4. **Accessibility**: Добавлены ARIA атрибуты и focus стили
5. **Performance**: Использованы CDN для Bootstrap и иконок
6. **SVG Graphics**: Кастомная SVG иллюстрация для hero секции
7. **Smooth Scroll**: Плавная прокрутка между секциями
8. **Interactive Effects**: Hover эффекты и анимации для улучшения UX

## Запуск

Откройте файл `landing.html` в браузере. Страница не требует сервера и работает локально.
